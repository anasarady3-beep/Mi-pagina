Cómo probar localmente:

1. Abrir `index.html` en tu navegador (doble clic).  
2. O iniciar un servidor local (recomendado) con Python:

```powershell
cd "C:\Users\anasa\OneDrive\Documentos\Pagina web"
python -m http.server 8000

# Luego abre http://localhost:8000/index.html
```
